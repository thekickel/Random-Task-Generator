# Random Task Generator - Генератор случайных задач

## Информация об авторе
**Имя и фамилия:** Артём Шаяхметов

**Дата создания:** Май 2026

## Описание программы
**Random Task Generator** - это GUI-приложение для генерации случайных задач с возможностью фильтрации по категориям. Приложение позволяет создавать собственные задачи, генерировать случайную задачу из списка, фильтровать по типу (учёба, спорт, работа), а также сохранять историю всех генераций.

### Возможности:
- ✅ Добавление новых задач с указанием названия и типа
- ✅ Генерация случайной задачи из имеющегося списка
- ✅ Фильтрация генерации по типу задачи (учёба, спорт, работа)
- ✅ Отображение истории всех сгенерированных задач с датой и временем
- ✅ Удаление задач из общего списка
- ✅ Очистка истории генераций
- ✅ Сохранение данных в JSON-файл
- ✅ Автоматическая загрузка данных при запуске
- ✅ Валидация вводимых данных (проверка на пустую строку)
- ✅ Предустановленные задачи для быстрого старта

## Инструкция по запуску
### Требования:
- Python 3.6 или выше
- Установленная библиотека Tkinter (обычно идёт в комплекте с Python)

### Запуск:
1. Откройте терминал или командную строку
2. Перейдите в директорию проекта: `cd random-task-generator`
3. Запустите приложение: `python main.py`

## Примеры использования

### Добавление новой задачи:
1. Введите название задачи в поле "Название"
2. Выберите тип задачи из выпадающего списка (учёба, спорт, работа)
3. Нажмите кнопку "Добавить"
4. Задача появится в общем списке задач

### Генерация случайной задачи:
1. Выберите фильтр по типу (или оставьте "Все")
2. Нажмите кнопку "Сгенерировать"
3. Случайная задача отобразится на экране и сохранится в истории

### Удаление задач:
- Выберите задачу в списке и нажмите "Удалить выбранное"
- Задача будет удалена из общего списка

### Управление историей:
- История всех генераций отображается в нижней части окна
- Каждая запись содержит дату, время и текст задачи
- Для очистки нажмите "Очистить историю" и подтвердите действие

## Тестовые сценарии:
1. Запустите приложение (загрузятся 9 предустановленных задач)
2. Добавьте новую задачу: "Посмотреть вебинар" (тип: учёба)
3. Установите фильтр "учёба" и нажмите "Сгенерировать"
4. Проверьте, что генерируются только учебные задачи
5. Добавьте ещё 2-3 задачи разных типов
6. Сгенерируйте несколько задач и проверьте историю
7. Закройте и откройте приложение - все данные сохранились

## Валидация данных
Приложение проверяет:
- Название задачи не может быть пустым
- При генерации с фильтром проверяется наличие задач нужного типа
- При удалении необходимо выбрать задачу из списка
- Очистка истории требует подтверждения
